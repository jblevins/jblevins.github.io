Welcome to Johnnie's Bookmarklets

This is a collection of bookmarklets intended to make working with CLO a little bit easier.

  • Perusable - reveals all reusables (other than those in attributes) on a page (requires s_debug=true)
  • Log Out - Logs you out of a CLO site. (Useful for sites that don't have login/logout in the header)
  • - Check jQuery version
  • Inject jQuery - if it's not already there.
  • Switch to Admin Console (if you have access)
