Welcome to Johnnie's Bookmarklets

This is a collection of bookmarklets intended to make working with CLO a little bit easier. Just drag to your bookmarks bar or copy the link and paste it into a new bookmark.

  • Perusable - reveals all reusables (other than those in attributes) on a page. If it doesn't find any reusables and doesn't see s_debug=true in the URL, it will offer to reload the page with s_debug=true. You'll then have to click it again. This bookmarklet is actually a small loader that injects the full script from github.

    (Source available at: https://github.com/jblevins/bookmarklets)

  • Perusable - self contained. This one is all here in the link. It isn't dependent on loading the full script from github, and wont be updated unless you do so manually.
  • Log Out - Terminates a user session on a CLO site. (Useful for sites that don't have login/logout in the header)
  • Check jQuery version - Does what it says on the tin. Also lets you know if $ is assigned to jQuery.
  • Inject jQuery - if it's not already there.
  • Switch to Admin Console (if you have access)
